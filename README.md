# Flex and Grid

This repository contains three HTML and CSS layout exercises created using **CSS Flexbox** and **CSS Grid**.

## Project Description

The project includes:

- **Task 1:** A multi-section webpage layout using CSS Grid.
- **Task 2:** A user profile card layout created with Flexbox.
- **Task 3:** A webpage layout containing a Header, Aside, Article, Images, and Footer using Grid and Flexbox.

## Technologies Used

- HTML5
- CSS3
- CSS Flexbox
- CSS Grid
